# 미국 항공편 운항 통계 데이터 분석

```
Usage :

  $SPARK_HOME/bin/spark-submit sparkflightdataanalyzer_2.11-0.1.jar {command} {input-file or year} {output-path}


Example :

  $SPARK_HOME/bin/spark-submit sparkflightdataanalyzer_2.11-0.1.jar departure 2008 /path/to/output

  or

  $SPARK_HOME/bin/spark-submit sparkflightdataanalyzer_2.11-0.1.jar departure /path/to/input-file /path/to/output
```
